Quick help

  • Drag widget headers to move widgets.
  • Resize any widget or picture from its edges or corners. The resize zones are hidden to keep the screen clean.
  • Shift-click: select more than one item.
  • Drag on empty board: select multiple items with a box.
  • Resize handles: drag any side or corner. If several items are selected, they resize together.
  • Class folder: put one .txt file per class, one name per line. Load the folder once, then choose a class.
  • Save layout: downloads a JSON file and also stores the latest layout in this browser.
